MQTT | Subscribe LoRa Device

Dokumentasi ini membantu Anda untuk mengambil data LoRa dari perangkat melalui protokol MQTT. Data dari perangkat LoRa anda dapat diambil dengan mengikuti tutorial ini . Jika Anda memerlukan bantuan atau menemukan informasi yang kurang, jangan ragu untuk menghubungi kami melalui support@antares.id

Prasyarat

  1. Memiliki software MQTT FX. Jika Anda belum memilikinya, silakan buka link berikut link. MQTT FX.
  2. Memiliki perangkat lora yang aktif dan mengirimkan data ke Platform ANTARES.

Langkah-langkah

  1. Pastikan sudah mengaktifkan perangkat lora dan juga sudah mengirimkan data ke Platform ANTARES. Jika belum, bisa mengikuti LoRa Tutorial
  2. Perangkat LoRa yang aktif akan mengirimkan data dengan format type uplink dan disertakan dengan datanya, seperti pada contoh pada gambar

  3. Salin access key pada akun yang ingin di-subscribe.

  4. Buka software MQTT FX. Antarmuka pengguna ditunjukkan pada gambar berikut.

  5. Pertama, Anda perlu mengonfigurasi agar Anda dapat terhubung dengan MQTT Broker kami. Konfigurasi ditunjukkan seperti dibawah ini.

  6. Ketika Anda berhasil terhubung, akan muncul ikon berwarna hijau pada sebelah kanan atas.

  7. Klik Subscribe, kemudian jalankan subscribe a topic dengan mengikuti format berikut:

    /oneM2M/resp/antares-cse/your-access-key/choose-a-serialization-format
    Parameter choose-a-serialization-format dapat mempunyai 2 nilai:

    • xml untuk format XML
    • json untuk format JSON
    Pada langkah ini berarti Anda akan mendapatkan pesan/notifikasi ketika terdapat suatu perubahan yang terjadi pada akun Anda. Pada tutorial ini, kita menggunakan MQTT FX tools untuk subscribe, tetapi pada skenario kasus nyata, Anda dapat menggunakan aplikasi Anda untuk mengambil data dari perangkat LoRa Anda.

  8. Seteleh mensubscribe topic dengan format topic yang benar, Anda cukup mengunggu hingga perangkat Anda mengirimkan data. Jika data sudah terikirim, maka akan muncul pada kolom di bagian kanan sperti pada gambar.

    9. Semua data didalam "con" sesuai dengan data yang dikirimkan dari device perangkat Anda. Anda dapat membuat data format milik Anda pada device LoRa milik anda.

    Selamat!! Anda baru saja mendapatkan request pertama dengan menggunakan MQTT API.
