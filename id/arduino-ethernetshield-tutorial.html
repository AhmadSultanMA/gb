Tutorial Arduino Uno Ethernet Shield

Dokumentasi ini membantu Anda untuk menggunakan Arduino Uno dengan Ethernet Shield dan Antares. Jika membutuhkan informasi lebih lanjut, jangan ragu hubungi kami melalui support@antares.id

    • Instalasi Arduino

Pendahuluan

Dokumentasi ini akan menjelaskan tutorial Arduino Uno untuk mengirimkan data sederhana ke Antares menggunakan ethernet shield.

Prasyarat

Sebelum memulai, pastikan Anda telah memenuhi syarat sebagai berikut:

  • Arduino IDE
    • Silahkan install Arduino IDE pada komputer Anda. Klik Instalasi Arduino IDE untuk petunjuk instalasi
  • Library Antares Ethernet Shield Arduino IDE
    • Silahkan download library Antares ethernet shield pada Arduino IDE dengan klik Download Library Antares Ethernet Shield
  • Buat Applikasi dan Device pada console Antares akun Anda



Instalasi Arduino IDE

Berikut langkah - langkah instalasi Arduino IDE

  1. Buka arduino.cc. Klik Software untuk mendapatkan installer.

  2. Pilih installer sesuai dengan Operating System yang Anda gunakan.

  3. Pilih "JUST DOWNLOAD" untuk memulai proses Download.

  4. Simpan installer pada laptop/personal computer.

  5. Pilih "I Agree" untuk memroses instalasi.

  6. Pilih "Next" untuk memroses instalasi.

  7. Pilih "Install" untuk memroses instalasi.

  8. Tunggu hingga proses instalasi selesai.

  9. Pilih "Install" untuk memroses instalasi.

Selamat, proses instalasi telah selesai!!



Instalasi Library Antares Ethernet Shield

  1. Sebelum memulai proses pada software, Anda perlu menginstall Library Antares Ethernet Shield pada Arduino IDE yang Anda gunakan. Pada Arduino IDE, buka add .ZIP Library dengan klik Sketch > Include Library > add .ZIP Library.

  2. Kemudian pilih file zip Library Antares Ethernet Shield pada direktori penyimpanan yang telah Anda download sebelumnya. Lalu klik Open.



Konfigurasi

Ikuti langkah - langkah berikut untuk memulai proses konfigurasi :

  1. Masuk pada laman console Antares akun Anda

  2. Salin nama application dan device yang Anda gunakan di Antares ke example source code send data library Arduino Uno Ethernet Shield



    1. Pada Arduino IDE, konfigurasikan Arduino Uno sebagai board yang akan Anda gunakan. Klik Tools > Board > Arduino Uno.
    2. Pada Arduino IDE, konfigurasikan Port sesuai dengan Board Arduino yang Anda gunakan. Klik Tools > Port > COM ....



Kirim Data Sederhana ke Antares

Berikut merupakan source code untuk menghubungkan Arduino Uno Ethernet Shield dengan Antares menggunakan Arduino IDE.

Setelah Anda selesai melakukan konfigurasi, saatnya membuat kode program untuk mengirim data sederhana ke device pada console Antares yang telah Anda buat sebelumnya. Kode Program dapat Anda dapatkan di direktori example pada folder library Antares Ethernet Shield yang telah Anda download sebelumnya atau dengan mudah Anda dapatkan dengan klik File > Examples > AntaresEthernetShield > Send_Data pada Arduino IDE.

Kode program AntaresEthernetShield seperti berikut. Jangan lupa untuk ketik Access Key applicationname dan Device Id yang Anda gunakan pada console Antares akun Anda. Setelah itu upload program dan tunggu beberapa detik.

#include "AntaresEthernetShield.h"

#define ACCESSKEY "6e82abaac2765cbb:ed353af4fc53c5cf"
#define applicationName "ArduinoUno_EthernetShield"
#define deviceName "Test"

unsigned int counter =0;

AntaresEthernetShield antares(ACCESSKEY);

void setup() {
Serial.begin(9600);
antares.StartConnection();
}
void loop() { 
String text ="Hello! ";
text += String(counter);
counter++;
antares.sendData(text, applicationName, deviceName);
delay(10000);
}
