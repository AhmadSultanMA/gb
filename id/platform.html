Antares Platform

Antares memberikan layanan IoT Platform untuk pengembangan solusi IoT baik perangkat device maupun aplikasi berbasiskan standard OneM2M.

Antares menyediakan protokol komunikasi berbasis HTTP, MQTT, Websoket, dan CoAP dengan format data JSON dan XML.

Antares menyediakan library dan panduan pengembangan solusi IoT untuk microcontroller terkemuka seperti Arduino, ESP, STMicro, dan lain-lain dengan sistem komunikasi GSM/GPRS, NBIoT, WiFi, LAN, dan LoRaWAN., selain menyediakan panduang pengembangan aplikasi mobile dan web.
