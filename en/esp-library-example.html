Introduction

This documentation will explain how an ESP8266 board can read data from some sensors, and then the data will be sent to the Antares IoT Platform. For more information, please download the Antares workshop module.

Antares ESP8266 Workshop Module (pdf)

Hardware

To measure temperature and light intensity, some electronic components are needed. Here are some of them:

  • ESP8266
    • The device to read sensor readings. The board WeMos D1 Mini will be used
  • DHT11
    • Temperature and humidity sensor
  • LDR
    • Light intensity sensor

Software

Here are the softwares which are needed to run some examples in this documentation:

  • Arduino IDE
    • The main software to program ESP8266.
    Download Arduino IDE
  • Arduino ESP8266
    • Arduino IDE does not support ESP8266 by default, so we need to install the board first.
    Download Arduino ESP8266
    The guide to install ESP8266 board to Arduino IDE is included in the Antares workshop module.
  • Antares ESP8266 Library
    • The communication between ESP8266 and Antares database can be formed through HTTP or MQTT.
    These libraries can be installed with Arduino IDE's Library Manager, or you can download it directly in the corresponding GitHub pages.
    • Antares ESP8266 HTTP
  • DHT11 Library
    • DHT11 library can be installed to Arduino IDE through the Library Manager or directly from the GitHub page.
    Download Adafruit DHT Library

Arduino ESP8266 Basic Program

This part has some examples to run basic program in ESP8266 with Arduino IDE.

Blink
Every ESP8266 board has a built-in LED which you can control by accessing the GPIO2 and giving HIGH or LOW logic.
Since the LED is active low, when you give it a HIGH logic, the LED wil turn off, and vice versa. 
#define led 2                 // Make "led" variable as GPIO2

void setup() {
  pinMode(led, OUTPUT);       // Make "led" pin as output
}

void loop() {
  digitalWrite(led, HIGH);    // Turn LED off
  delay(1000);                // 1 second delay
  digitalWrite(led, LOW);     // Turn LED On
  delay(1000);                // 1 second delay
}

GPIO
GPIO in ESP8266 boards are usually preceded by D, for example D4, D5, etc.
You can take the circuit below as the reference to run this example (Note that the resistor is around 220 ohms). 
#define led1 D2               // Make "led1" as D2
#define led2 D4               // Make "led2" as D4

void setup() {
  pinMode(led1, OUTPUT);      // Set "led1" as output
  pinMode(led2, OUTPUT);      // Set "led2" as output
}

void loop() {
  digitalWrite(led1, HIGH);   // Turn led1 on
  digitalWrite(led2, LOW);    // Turn led2 off
  delay(1000);                // 1 second delay
  digitalWrite(led1, LOW);    // Turn led1 off
  digitalWrite(led2, HIGH);   // Turn led2 on
  delay(1000);                // 1 second delay
}

Serial Monitor
Serial monitor can be used to send data from ESP8266 to your computer's USB port so that the data will be sent to your computer. 
void setup() {
  Serial.begin(115200);     // Set serial baudrate to 115200
}

int num = 0;                // Initiate "num" variable as integer

void loop() {
  Serial.print("Num: ");    // Send the word "Num: your-number" to the serial monitor
  Serial.println(num);
  num++;                    // Increase num by 1
  delay(1000);              // 1 second delay
}

Sensor Temperatur
You'll need a DHT library to run this example.
You can take the circuit below as a reference to run this example; 
#include "DHT.h"            // DHT11 library initiation
#define DHTPIN D2           // Make DHTPIN variable as D2
#define DHTTYPE DHT11       // Set DHT type as DHT11
DHT dht(DHTPIN, DHTTYPE);   // Make dht object

void setup() {
	Serial.begin(115200);    // Set baudrate as 115200
	dht.begin();             // Initiate dht object
}

void loop() {
	float hum = dht.readHumidity();      // Read humidity
	float temp = dht.readTemperature();  // Read temperature
	if (isnan(hum) || isnan(temp)) {     // Check if temp and hum data is valid
      Serial.println("Failed reading DHT sensor!");
      return;
	}
	Serial.println("Humidity: " + (String)hum + " %");       // Print hum (%)
	Serial.println("Temperature: " + (String)temp + " *C");  // Print temp (*C)
	delay(1000);
}

Antares HTTP

Below are examples for exchanging data through HTTP between ESP8266 and Antares database.

Send Data
The example code is in File > Examples > Antares ESP8266 HTTP > AntaresStoreData 
#include <AntaresESP8266HTTP.h>     // Initiate Antares HTTP Library

#define ACCESSKEY "your-access-key"       // Put your Antares account key here
#define WIFISSID "your-wifi-ssid"         // Put your SSID here
#define PASSWORD "your-wifi-password"     // Put your WiFi password here

#define applicationName "your-application-name"   // Put your application name here
#define deviceName "your-device-name"     // Put your device name here

AntaresESP8266HTTP antares(ACCESSKEY);    // Make antares object

void setup() {
  Serial.begin(115200);     // Set serial baudrate to 115200
  antares.setDebug(true);   // Set debug as true. Set to false if you don't want to show Antares messages to serial monitor
  antares.wifiConnection(WIFISSID,PASSWORD);  // Try to connect to WiFi
}

void loop() {
  // Put random variables
  int temp = random(25,30);
  int hum = random(75,90);
  float windsp = float(random(20, 30))/3.33;
  float rainlv = float(random(0, 20))/6.99;
  String lat = "-6.8718189";
  String lon = "107.5872477";

  // Put these defined variables to a storage buffer in ESP8266
  antares.add("temperature", temp);
  antares.add("humidity", hum);
  antares.add("wind_speed", windsp);
  antares.add("rain_level", rainlv);
  antares.add("latitude", lat);
  antares.add("longitude", lon);

  // Send from buffer to ESP8266
  antares.send(applicationName, deviceName);
  delay(10000);
}

Get Latest Data
This code will fetch the latest data form your Antares application device.
The example code is in File > Examples > Antares ESP8266 HTTP > AntaresGetLatestData 
#include <AntaresESP8266HTTP.h>     // Initiate Antares HTTP Library

#define ACCESSKEY "your-access-key"           // Put your Antares account key here
#define WIFISSID "your-wifi-ssid"             // Put your SSID here
#define PASSWORD "your-wifi-password"         // Put your WiFi password here

#define applicationName "your-application-name"       // Put your application name here
#define deviceName "your-device-name"         // Put your device name here

AntaresESP8266HTTP antares(ACCESSKEY);        // Make antares object

void setup() {
  Serial.begin(115200);                       // Set serial baudrate to 115200
  antares.setDebug(true);                     // Set debug as true. Set to false if you don't want to show Antares messages to serial monitor
  antares.wifiConnection(WIFISSID,PASSWORD);  // Try to connect to WiFi
}

void loop() {
  // Get latest data from Antares
  antares.get(applicationName, deviceName);

  // Get individual data
  int temp = antares.getInt("temperature");
  int hum = antares.getInt("humidity");
  float windsp = antares.getFloat("wind_speed");
  float rainlv = antares.getFloat("rain_level");
  String lat = antares.getString("latitude");
  String lon = antares.getString("longitude");

  // Send to serial monitor
  Serial.println("Temperature: " + String(temp));
  Serial.println("Humidity: " + String(hum));
  Serial.println("Wind speed: " + String(windsp));
  Serial.println("Rain level: " + String(rainlv));
  Serial.println("Latitude: " + lat);
  Serial.println("Longitude: " + lon);
  delay(10000);
}

Antares MQTT

Below are the examples to exchange data betwen ESP8266 and Antares through the MQTT protocol.
Subscribing data is real-time. Which means that every time a data is sent to your Antares device, it will immediately be received to your ESP8266.

Send (Publish)
The example to publish data is in File > Examples > Antares ESP8266 MQTT > AntaresMQTTPublish 
#include <AntaresESP8266MQTT.h>     // Initiate Antares MQTT library

#define ACCESSKEY "your-access-key"           // Put your Antares account key here
#define WIFISSID "your-wifi-ssid"             // Put your SSID here
#define PASSWORD "your-wifi-password"         // Put your WiFi password here

#define applicationName "your-application-name"       // Put your application name here
#define deviceName "your-device-name"         // Put your device name here

AntaresESP8266MQTT antares(ACCESSKEY);        // Make antares object

void setup() {
  Serial.begin(115200);                       // Set serial baudrate to 115200
  antares.setDebug(true);                     // Set debug as true. Set to false if you don't want to show Antares messages to serial monitor
  antares.wifiConnection(WIFISSID,PASSWORD);  // Try to connect to WiFi
  antares.setMqttServer();                    // Initiate Antares MQTT server
}

void loop() {
  /*
    Check the connection to the Antares MQTT broker.
    If disconnected, will try to reconnect.
   */
  antares.checkMqttConnection();

  // Initiate random values
  int temp = random(25,30) ;
  int hum = random(75,90);
  float windsp = float(random(20, 30))/3.33;
  float rainlv = float(random(0, 20))/6.99;
  String lat = "-6.8718189";
  String lon = "107.5872477";

  // Put the variables to buffer
  antares.add("temperature", temp);
  antares.add("humidity", hum);
  antares.add("wind_speed", windsp);
  antares.add("rain_level", rainlv);
  antares.add("latitude", lat);
  antares.add("longitude", lon);

  // Publish from buffer to Antares
  antares.publish(applicationName, deviceName);
  delay(5000);
}

Listen (Subscribe)
The example to subscribe to incoming MQTT message is in File > Examples > Antares ESP8266 MQTT > AntaresMQTTSubscribe 
#include <AntaresESP8266MQTT.h>     // Initiate Antares MQTT library

#define ACCESSKEY "your-access-key"           // Put your Antares account key here
#define WIFISSID "your-wifi-ssid"             // Put your SSID here
#define PASSWORD "your-wifi-password"         // Put your WiFi password here

#define applicationName "your-application-name"       // Put your application name here
#define deviceName "your-device-name"         // Put your device name here

AntaresESP8266MQTT antares(ACCESSKEY);        // Make antares object

/*
  The main function to get topic and payload
  as soon as an MQTT mesage is received in your ESP8266
*/
void callback(char topic[], byte payload[], unsigned int length) {

  antares.get(topic, payload, length);  // Keep topic and payload in buffer

  Serial.println("New Message!");
  Serial.println("Topic: " + antares.getTopic());     // Print topic
  Serial.println("Payload: " + antares.getPayload()); // Print payload

  // Print individual data
  Serial.println("Temperature: " + String(antares.getInt("temperature")));
  Serial.println("Humidity: " + String(antares.getInt("humidity")));
  Serial.println("Wind speed: " + String(antares.getFloat("wind_speed")));
  Serial.println("Rain level: " + String(antares.getFloat("rain_level")));
  Serial.println("Latitude: " + antares.getString("latitude"));
  Serial.println("Longitude: " + antares.getString("longitude"));
}

void setup() {
  Serial.begin(115200);                       // Set serial baudrate to 115200
  antares.setDebug(true);                     // Set debug as true. Set to false if you don't want to show Antares messages to serial monitor
  antares.wifiConnection(WIFISSID,PASSWORD);  // Try to connect to WiFi
  antares.setMqttServer();                    // Initiate Antares MQTT server
  antares.setCallback(callback);              // Set callback function as defined above
}
void loop() {
  /*
    Check the connection to the Antares MQTT broker.
    If disconnected, will try to reconnect.
   */
  antares.checkMqttConnection();
}
