  • Create an Application

    Easily create an application through the dashboard.
  • Add Your Devices

    Add as much devices under your application as you want. Determine whether that is a sensor or an actuator.
  • Run Your App Server

    Create your own login on your server. We will notify you for any changes of an application or device.

Support Multiple Protocols

There is a bunch of communication protocols that will support your devices. No reason for incompatibility.

HTTP

HTTP is supported by us. As a mature Internet protocol, HTTP is one of the most protocol use nowadays. IoT device and HTTP are two unseparated entities.

MQTT

MQTT is a perfect match with IoT devices as it's extremely lightweight. It is useful for connections with remote locations where a small code footprint is required and/or network bandwidth is at a premium.

COAP

The Constrained Application Protocol (CoAP) is a specialized web transfer protocol for use with constrained nodes and constrained networks in the IoT.
