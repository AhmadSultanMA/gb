Introduction

MIT App Inventor is a platform to facilitate the process of making simple applications without having to learn or use too many programming languages. We can design Android applications as desired by using various layouts and components available.

Below is how to use MIT App Inventor. If you need help and find flaws in this tutorial, can contact us via support@ANTARES.id


Display of MIT App Inventor

MIT App Inventor has two main pages, namely designer page and blocks page. The designer page is used to design the appearance of the application with various components and layouts provided as desired. While blocks page are used to program the course of the Android application in accordance with the purpose.

Figure 1. Display of Designer Page

Figure 2. Display of Blocks Page

Designer Page

In the designer page there are several windows like Palette, Viewer, Components, Media, and Properties. These tools function to design the appearance of the Android application as desired.

  1. Palettete is a window where components are categorized into several categories to be included in the application created. There are categories of User Interface, Layout, Media, Drawing and Animation, Maps, Sensors, Social, Storage, Connectivity, LEGO MINDSTORMS, Experimental, and Extension.
  2. Viewer is a place to adjust the appearance of components in the application.
  3. Components are places to manage components that have been placed in the viewer, such as changing the name of a component, and deleting a component.
  4. Properties is a place to set layer properties, and the components used in applications that are being made such as width, height, background color, font size, etc.
  5. Media is a place to upload images for use in applications that are being created.
